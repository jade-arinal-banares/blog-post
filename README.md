### How to run the project?

1. Go to terminal and type the following command

2. To download node module dependencies
```
  npm install
```

3. Run the server
```
  npm start or yarn start
```
```
  npm install -g npm@9.2.0
```

4. For production
```
  npm run build or yarn build
```
